# 用語集

[system_specification.md](./system_specification.md)・[designs/以下の設計書](../designs/)・コード上のコメントで表記揺れが起きないよう、一意に定めておくべき用語をここにまとめる。新しい用語を追加する場合や、既存の用語の使い方に迷った場合はこのファイルを参照する。

| 用語 | 定義 | 対応するコード上の識別子（参考） |
| --- | --- | --- |
| 自転車ログ | Stravaと連携して取得・表示する自転車走行記録の機能・レイヤー全体を指す総称（例:「自転車ログ表示機能」「自転車ログレイヤー」）。個々の走行記録は「アクティビティ」と呼ぶ | `bicycle-log`（レイヤーID）、`BICYCLE_LOG_*`定数 |
| アクティビティ | Stravaの1回の走行記録に対応するデータ1件。地図上では1本の線（軌跡）として描画される | `CyclingActivity`（フロントエンド）、`CyclingActivityEntity`（バックエンド） |
| 同期 | レイヤONのタイミングで、前回同期時刻以降にStravaへ追加された新規アクティビティのみを取得しDBへ反映する処理。「更新系API」「更新用API」等の別表現は使わず「同期」に統一する | `sync()`（`ActivitiesService`）、`syncCyclingActivities`（フロントエンド） |
| 初期取り込み（バックフィル） | DB上にまだ詳細を取得していない全アクティビティを、Strava APIのレート制限に従いながら非同期に取得しDBへ保存する機能。コード上は英語で「バックフィル」と呼ぶため、仕様書・ユーザー向け表示では「初期取り込み」、コード識別子では「バックフィル」を使う | `ActivitiesBackfillService`、`startBackfill`、`BackfillStatus` |
| 自転車ログ強制再取得 | 初期取り込みとは別に、DB上の既存全アクティビティの詳細を取得し直す機能（新規アクティビティの検出は行わない） | `startForceRefetch` |
| マップコントロール | 地図右下に浮かぶ、レイヤー切り替え・フィルタ・設定の各ダイアログを開くアイコンボタン群（Issue #32で左サイドバーを廃止し導入） | `MapControls`コンポーネント |
| レイヤーダイアログ | マップコントロールのレイヤーアイコンから開く、レイヤーの表示/非表示を切り替えるダイアログ | `LayerDialog`コンポーネント |
| 設定ダイアログ | マップコントロールの設定アイコンから開く、自転車ログ初期取り込み・強制再取得を開始するダイアログ | `SettingsDialog`コンポーネント |
| 進捗フッター | 地図下部に表示される、初期取り込み・強制再取得の進捗（実行中）・完了状態を表示する領域 | `BackfillProgressFooter`コンポーネント |
| 右サイドバー | 選択中アクティビティの一覧・詳細を表示する、画面右側の領域 | `ActivityDetailSidebar`コンポーネント |
| 選択（選択中） | 地図上のアクティビティの線をクリックして検出された状態。右サイドバーに一覧表示される対象。フォーカスとは独立した状態で、複数件を保持できる | `selectedIds`、`useActivitySelection` |
| フォーカス（フォーカス中） | 選択中一覧から1件をクリックして詳細画面を表示している状態。選択中の集合のうち1件を指す | `focusedIndex`/`focusedId`、`useActivitySelection` |
| 一覧画面 | 右サイドバーの、選択中アクティビティ一覧（走行開始日時の通し番号付きリスト）を表示する状態 | `ActivityList`コンポーネント |
| 詳細画面 | 右サイドバーの、フォーカス中アクティビティの詳細情報を表示する状態 | `ActivityDetail`コンポーネント |
| 通過自治体 | アクティビティの軌跡が通過した市区町村（都道府県名+市区町村名）の一覧 | `PassedMunicipality`、`MunicipalitiesService` |
| スタート・ゴールマーカー | フォーカス中アクティビティの軌跡の開始地点・終了地点に表示するアイコン（スタート:再生アイコン、ゴール:旗アイコン） | `createStartMarkerElement`/`createGoalMarkerElement` |
| フィルタ | 地図上に表示する自転車ログを、年月・獲得標高・平均時速・走行距離の条件で絞り込む機能 | `ActivityFilter`、`filterActivities`、`FilterDialog` |
| エラーダイアログ | APIエラー等が発生した際に、エラーメッセージとユーザーが取るべき対応（ヒント）を表示するダイアログ | `ErrorDialog`コンポーネント |
